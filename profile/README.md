# Traveller
### By Pietro Bossolasco

### Cos'è Traveller?
Traveller è una piattaforma che permette di organizzare viaggi da soli o con i propri amici in modo estremamente facile!
In oltre la piattaforma permette di prendere ispirazione di propri amici vedendo i propri viaggi.

### Struttura progetto
La struttura dei branch si presenta:
* App (dev e release)
* Server / Versione web (dev e release)
* ui e ux (Progettazione delle interfacce tramite XD)

### Tecnologie utilizzate
Per la creazione del progetto viengono utilizzati:
- Mobile:
    * React Native
    * Axios
    * MUI
- Server:
    * Mongo DB
    * TypeScript
- Web:
    * Tailwind CSS
- UX e UI:
    * Adobe XD
